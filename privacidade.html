Política de Privacidade

Última atualização: 23 de março de 2026

1. Introdução

A Dra. Júnia Nascimento ("nós") respeita a privacidade dos visitantes deste site e está comprometida com a proteção dos seus dados pessoais, em conformidade com a Lei Geral de Proteção de Dados Pessoais (LGPD — Lei nº 13.709/2018).

2. Dados coletados

Podemos coletar os seguintes dados pessoais, fornecidos voluntariamente por você através do formulário de contato:

  • Nome completo
  • Telefone / WhatsApp
  • E-mail
  • Tipo de consulta desejada
  • Motivo da consulta (informação de saúde — dado sensível)

Importante: Os dados inseridos no formulário de agendamento são enviados diretamente para o WhatsApp da Dra. Júnia Nascimento. Este site não armazena, processa ou retém nenhum dado pessoal em servidores próprios.

3. Finalidade do tratamento

Os dados coletados são utilizados exclusivamente para:

  • Viabilizar o agendamento de consultas médicas
  • Entrar em contato para confirmação de horários
  • Prestar atendimento médico solicitado

4. Base legal

O tratamento dos dados pessoais se baseia no consentimento do titular (Art. 7º, I da LGPD) ao preencher e enviar o formulário de contato, e na tutela da saúde (Art. 7º, VIII) quando os dados são utilizados para fins de atendimento médico.

5. Compartilhamento de dados

Seus dados pessoais não são compartilhados com terceiros, exceto:

  • WhatsApp (Meta): utilizado como canal de comunicação para agendamento
  • Cloudflare: hospedagem do site e analytics anônimos de navegação (sem coleta de dados pessoais)

6. Cookies e analytics

Este site utiliza o Cloudflare Web Analytics, que coleta dados anônimos de navegação (páginas visitadas, país de origem, tipo de dispositivo) sem uso de cookies e sem identificar visitantes individualmente. Nenhum cookie de rastreamento é instalado no seu navegador.

7. Dados sensíveis de saúde

O campo "Motivo da consulta" pode conter informações de saúde, consideradas dados pessoais sensíveis pela LGPD. Estas informações são protegidas pelo sigilo médico (Art. 73 do Código de Ética Médica) e tratadas com o mais alto nível de confidencialidade.

8. Seus direitos (LGPD)

Como titular dos dados, você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a eliminação dos dados pessoais
  • Revogar o consentimento a qualquer momento

Para exercer qualquer desses direitos, entre em contato pelo e-mail junianascimentohemato@gmail.com.

9. Segurança

Este site utiliza conexão criptografada (HTTPS/SSL) fornecida pelo Cloudflare, garantindo que os dados trafeguem de forma segura entre o seu navegador e nossos servidores.
